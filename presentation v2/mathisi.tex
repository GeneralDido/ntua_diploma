explained\_variance(y,\widehat{y}) = 1 - \frac{Var\left \{ y -\widehat{y}\right \}}{Var\left \{ y \right \}}
\end{equation}

\frametitle{Μέθοδος Ανάλυσης - Μοντέλα}
\begin{frame}
\frametitle{Σταθερότητα Μοντέλων}
Θέλουμε τα μοντέλα να είναι σταθερά ως προς την δομή τους:
\begin{itemize}
\item Να μην έχουν ακραία χαρακτηριστικά (όπως πολύ χαμηλή ακρίβεια) σε κάποια από τις εκτελέσεις
\item Παρόμοιες σημαντικότητες μεταβλητών σε διαφορετικές εκτελέσεις
\end{itemize}
\end{frame}
